# My Portfolio

A personal portfolio website built with React and Tailwind CSS to present my profile, technical skills, selected software development projects, certificates, and contact information.

## Overview

The main goal of this project is to present:

- My background and career transition into software development
- My technical skills and learning direction
- Selected projects that demonstrate backend logic, maintainable architecture, and practical software thinking beyond UI
- My ability to work primarily on backend development while also supporting frontend implementation when needed
- Certificates from Generation Thailand and Codecademy
- Resume and contact information in one place
